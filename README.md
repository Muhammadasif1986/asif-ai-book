# Asif AI Book

Comprehensive guide and framework for AI development with practical examples and documentation.

## Documentation

The project includes comprehensive documentation built with Docusaurus, located in the `my_book/` directory:

- **Documentation Site**: Complete guides, tutorials, and API documentation
- **Location**: `my_book/` directory
- **Framework**: Docusaurus v3.x
- **Content**: AI fundamentals, practical implementations, and best practices

To run the documentation site locally:

```bash
cd my_book
npm install
npm start
```

## Project Structure

- `my_book/` - Docusaurus documentation site
- `backend/` - Backend services and APIs
- `specs/` - Project specifications and plans
- `.specify/` - Spec-driven development tools and templates

## Features

- Comprehensive AI development documentation
- Practical examples and tutorials
- API documentation and guides
- Best practices for AI development
- Modular architecture for easy extension
